\doxysubsection{Detailed Description}
To communicate with the server, BaseX uses a websocket. In the server protocol for BaseX, 2 sub-\/protocols are distinguished\+:
\begin{DoxyItemize}
\item The command protocol applies to commands sent to the server. The Query command is one of the commands. This command is used to create a Query object.
\item The query command protocol applies to commands sent to the Query object. 
\end{DoxyItemize}
