\doxysubsection{Detailed Description}
\doxyref{Basex\+Client}{p.}{classBasexCpp_1_1BasexClient}

Each client provides a session class or script with methods to connect to and communicate with the database server. A socket connection will be established by the constructor, which expects a host, port, username and password as arguments. The port number can be offered as a string or as an integer.
